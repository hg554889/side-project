# Side Project

React.js + Express.js 풀스택 애플리케이션

## 프로젝트 구조

```
side-project/
├── client/          # React.js (Vite)
└── server/          # Express.js
```

## 실행 방법

### 클라이언트 실행
```bash
cd client
npm run dev
```

### 서버 실행
```bash
cd server
npm run dev
```

## 기술 스택

- **Frontend**: React.js with Vite
- **Backend**: Express.js
